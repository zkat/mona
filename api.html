<static> parseStream(parser, opts) → {stream.Transform}

Implements node.js' transformer stream API. That is, it implements the transform stream interface.

By default, parseStream is a passthrough stream for incoming data, and a "parsed" event is emitted on the results of successful parses. If your parser returns string/binary chunks, you can use the transformOutput option to make parseStream's output be the parser's "data" output instead of just the incoming data.

For now, parseStream only supports utf8 streams.
